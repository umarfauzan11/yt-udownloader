# YouTube Downloader

Web app untuk download video & MP3 dari YouTube.

## Tech Stack
- **Frontend**: Next.js 16 + React 19 + Tailwind CSS
- **Backend**: Node.js + Express + yt-dlp
- **Deploy**: Netlify (frontend) + Railway (backend)

## Local Development
```bash
# Terminal 1: Backend
cd server
npm install
node index.js

# Terminal 2: Frontend
cd client
npm install
npm run dev
