# YouTube UDownloader

Web app for download Video & Mp3 from YouTube, can run local on your device with clone this repo.

## Tech Stack
- **Frontend**: Next.js 16 + React 19 + Tailwind CSS
- **Backend**: Node.js + Express + yt-dlp
- **Deploy**: Netlify (frontend) + Railway (backend)

## Local Development
```bash
# Terminal 1: Backend
cd server
npm install

# Terminal 2: Frontend
cd client
npm install
```

## Running Local
```bash
npm run dev (On the root, it starts two services: one running on the server and one running on the client.)
